## üöÄ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.) - optional, for editing

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/royal-mgwasi-hotel.git
```

2. Navigate to the project directory
```bash
cd royal-mgwasi-hotel
```

3. Open `index.html` in your web browser
```bash
# On Windows
start index.html

# On macOS
open index.html

# On Linux
xdg-open index.html
```

Or simply double-click the `index.html` file to open it in your default browser.

## üé® Customization

### Updating Colors
Modify the CSS variables in `css/style.css`:
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  /* Add more custom colors */
}
```

### Adding New Pages
1. Create a new HTML file
2. Link the CSS and JS files
3. Update navigation menu in all pages
4. Add corresponding styles if needed

### Modifying Content
- Update text content directly in HTML files
- Replace images in the `assests/` directory
- Adjust styling in CSS files
